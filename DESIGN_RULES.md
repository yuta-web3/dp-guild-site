# 基本思想

> **「判断を邪魔しないデザイン」**

- 目立たせない
- 説明しすぎない
- 気取らない
- 余白を恐れない

---

# AIO / AI視点の原則（重要）

## なぜ必要か

このサイトは **AIOで拾われることが最重要目標**。
デザインがAIの文脈理解を邪魔してはいけない。

## 原則

### 1. 見出しは単独で意味が成立する文にする

OK:
- 「何が正解か」を判断できていないことです。
- 最初の判断を一緒に決める会社です。

NG:
- 問題の本質
- 私たちについて

### 2. 装飾で文意を変えない

色やアイコンに意味を持たせない。AIは色を読めない。

OK:
- 向いている方：WebやSNSに課題は感じているが...（テキストで明示）

NG:
- 緑アイコンだけで「良い」を表現
- 赤色だけで「ダメ」を表現

### 3. 箇条書きは「判断基準」として読める形にする

OK:
- WebやSNSに課題は感じているが、何から始めるべきか分からない
- 施策の前に、考え方や優先順位を整理したい

NG:
- 課題あり
- 整理したい

### 4. 文の途中で改行しすぎない

AIが文を分断して解釈するリスクがある。

OK:
- 情報は溢れている。でも、自社にとって何が正解かは誰も教えてくれない。

NG:
- 情報は / 溢れている。/ でも、自社にとって / 何が正解かは / 誰も教えてくれない。

# 装飾の使用原則

## 基本姿勢

装飾は「使わない」ではなく **「意味がある時だけ使う」**。

## 使用してよい条件

装飾や視覚要素は、以下の場合のみ使用可：

1. **判断を早める場合** — 向いている/向いていないの○×マーク
2. **理解を助ける場合** — プロセスのステップ番号
3. **階層を明示する場合** — セクション背景色の切り替え

## 使用してはいけない条件

1. 「なんとなく寂しいから」
2. 「他社サイトがやってるから」
3. 「デザイン的に映えるから」

**迷ったら使わない。**

## 使っちゃダメな色

- 原色全般

# タイポグラフィ（固定）

## フォント

```css
font-family: var(--font-geist-sans), Arial, Helvetica, sans-serif;
```

**変更禁止。** デザインフォント導入NG。

## サイズスケール

| 用途 | サイズ | Tailwind |
|------|--------|----------|
| ヒーロー見出し | 48px (mobile: 30px) | `text-3xl md:text-5xl` |
| セクション見出し | 36px (mobile: 24px) | `text-2xl md:text-4xl` |
| 小見出し | 24px (mobile: 20px) | `text-xl md:text-2xl` |
| 本文 | 18px | `text-lg` |
| 補足テキスト | 16px | `text-base` |
| 注釈 | 14px | `text-sm` |

## 太字の使用

- 見出しのみ `font-bold`
- 本文中の強調は **1セクション1箇所まで**
- 乱用禁止

# 余白・スペーシング（固定）

## セクション間

| 箇所 | 値 | Tailwind |
|------|-----|----------|
| セクション上下 | 96px (mobile: 96px) | `py-24 md:py-32` |
| コンテンツ最大幅 | 768px | `max-w-3xl` |
| 広めのコンテンツ | 896px | `max-w-4xl` |

## 要素間

| 箇所 | 値 | Tailwind |
|------|-----|----------|
| 見出し下 | 32px | `mb-8` |
| 段落間 | 16-24px | `mb-4` ~ `mb-6` |
| リスト項目間 | 16px | `space-y-4` |

**余白を減らす変更は原則禁止。**

# ボタン・CTA（固定）

## スタイル

```jsx
// 標準CTA
className="inline-flex items-center px-8 py-4 bg-[#0F172A] text-white rounded-lg font-medium hover:bg-[#1e293b] transition-colors"

// 大きいCTA（最下部用）
className="inline-flex items-center px-10 py-5 bg-white text-[#0F172A] rounded-lg font-bold text-lg hover:bg-gray-100 transition-colors"
```

## 禁止事項

- 影（box-shadow）追加
- グラデーション
- アニメーション（hover以外）
- 常時追従（sticky/fixed）
- 派手な色

## アイコン

- `ArrowRight` のみ使用可
- サイズは `w-5 h-5` または `w-6 h-6`
- 他のアイコン追加は要検討

# 変更時チェックリスト

デザイン変更前に確認：

- [ ] 色はパレット内か
- [ ] フォントは変更していないか
- [ ] 余白は減っていないか
- [ ] 影・グラデーション・アニメーションを追加していないか
- [ ] 画像・イラストを追加していないか
- [ ] CTAが派手になっていないか
- [ ] 情報量が増えていないか
- [ ] 見出しは単独で意味が通るか（AIO）
- [ ] 装飾で文意を変えていないか（AIO）

**1つでもNGなら、その変更はしない。**

---

# 例外を許可する場合

以下の場合のみ、ルール外の変更を検討可：

1. A/Bテストで明確に効果が出た場合
2. ユーザーからの具体的なフィードバックがあった場合
3. 問い合わせ数が明らかに減少した場合

**「なんとなくダサいから」は変更理由にならない。**
