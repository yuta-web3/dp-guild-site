最終更新: 2026-01-09（構造化データ追加）

## 1. AIOとは

**AI Overview（旧SGE: Search Generative Experience）** は、Googleが検索結果の上部にAI生成の要約を表示する機能。

### 重要な事実
- Googleは **AIO最適化の公式ガイドラインを公開していない**
- 「AIO最適化」として語られる内容の多くは **推測や仮説**
- 効果が実証されている施策は限定的

## 2. 本サイトで実施したこと

### 2-1. コンテンツ設計（効果：中〜高）

#### 実施内容
| 施策 | 実装場所 | 根拠レベル |
|------|----------|------------|
| 「判断会社」という明確なポジショニング | 全ページ | 推奨（差別化） |
| 具体的な判断プロセスの公開 | /results | 推奨（E-E-A-T） |
| 「○○ではない、○○だ」の対比構造 | TOP, /about | 仮説（引用されやすい形式） |
| 結論→理由→具体例の構造 | ブログ記事 | 推奨（SEO基本） |

#### 具体例
```
// TOP ヒーローコピー
「何から手をつければいいかわからない」
そんな中小企業のための会社です。

// 定義文
作る会社ではありません。
最初の判断を一緒に決める会社です。
```

### 2-2. 判断ログ形式（/results）

#### 実施内容
6件の判断事例を以下の構造で記載：
```
・相談内容：「○○で迷っている」
・検討した選択肢：A / B / C
・やらなかった判断：B（理由）
・最終判断：Aを選択
・理由：○○
・その後：○○の結果
```

#### 根拠レベル
- **推奨**: 具体的な事実ベースのコンテンツはE-E-A-Tに寄与
- **仮説**: この形式がAIOに特に有効かは未実証

### 2-3. ブログ記事のメタタグ

#### 実施内容
```tsx
// app/blog/[id]/page.tsx
export async function generateMetadata({ params }) {
  return {
    description: blog.description || blog.aio_answer,
    other: blog.aio_answer ? { 'aio-answer': blog.aio_answer } : {},
  };
}
```

#### 根拠レベル
- **実験的**: `aio-answer` メタタグはGoogleの公式仕様ではない
- **効果不明**: Googleがこのメタタグを読んでいる証拠なし

### 2-4. 文言設計

#### 削除したもの（営業感排除）
| 削除した表現 | 理由 |
|--------------|------|
| 「最適なソリューションを提供」 | 抽象的・営業的 |
| 「豊富な実績」 | 具体性がない |
| 「高い顧客満足度」 | 数値なし |
| Impact Numbers（1500万再生等） | 制作会社に見える |
| 選ばれる理由（3枚カード） | 営業トーク |

#### 採用した表現
| 採用した表現 | 理由 |
|--------------|------|
| 「何から手をつけるかを一緒に決める」 | 具体的 |
| 「継続契約の解約 0件」 | 事実ベース |
| 「判断と改善を繰り返す関係」 | プロセス説明 |

### 2-5. ナビゲーション設計

#### 実施内容
```
[DP-GUILD] TOP | 実績・事例 | 会社概要 | ブログ | [相談する]
```

#### 削除した項目
- ソリューション → 制作会社に見える
- 料金 → 判断支援に定価はない
- サービス一覧 → 何でも屋に見える

### 2-6. UX改善（損失回避バイアス対応）

#### 実施内容
```tsx
// TOP CTAセクション
<p className="text-white/50 text-sm mt-6">
  ※ 相談後に「今は何もしない」という結論になることもあります
</p>
```

#### 根拠レベル
- **推奨**: 行動経済学に基づくCV改善施策

## 8. 注意事項

**このドキュメントに記載された「AIO最適化」施策の多くは、効果が実証されていない仮説に基づいています。**

Googleは検索アルゴリズムを公開していないため、「これをやればAIOに載る」という確実な方法は存在しません。

本サイトの設計は、AIO対策よりも以下を優先しています：
1. **ターゲットユーザーにとって有用なコンテンツ**
2. **「判断会社」としての一貫したポジショニング**
3. **事実ベースの具体的な表現**

これらは結果的にAIOに選ばれやすい可能性がありますが、それは副次的な効果です。
